Ceci est un petit plugin pour Zibase ultra simple à utiliser. Je tiens a remercier Baptiste MARIENVAL de qui je me suis inspiré pour ce plugin.

Son fonctionnement est extrêmement simple puisque d'une simple commande "SARAH mets à jour la domotique", il va automatiquement recupérer l'ensemble des modules et scénarios de votre Zibase et enrichir le vocabulaire.

1. CONFIGURATION XIBASE.PROP

Le fichier xibase.prop permet de paramétrer les informations nécessaires au bon fonctionnement du plugin.
Il est donc indispensable de renseigner ces 5 éléments :

  • plateforme_web : adresse de la plateforme zibase de votre installation (par ex. zibase.net ou zibase2.net)
  • ip_lan : adresse IP locale de votre zibase. Se trouve dans le configurateur zibase dans la section "identifiants de Zibase".
  • API_version : 1 ou 2. Par defaut ou si vous ne savez pas, laisser ce paramêtre sur "1".
  • device : nom de votre Zibase. Se trouve dans le configurateur zibase dans la section "identifiants de Zibase".
  • token : token associé Zibase. Se trouve dans le configurateur zibase dans la section "identifiants de Zibase".

    2. IMPORTATION DES PERIPHERIQUES

    cette phase est essentielle, elle permet au plugin d'enrichir son vocabulaire en fonction de votre propre configuration et sans manipulation particuliaire.
  • Dites simplement :"SARAH mets a jour ma domotique". Et VOILA !!


    3. LE FICHIER ZIBASE.XML

    Une fois l'importation effectuée, tous les periphériques VISIBLES DEPUIS L'INTERFACE sont importés dans la section "Automatic generation".
  • vous pouvez alors modifier certain éléments de cette section, à savoir :
    • <item> qui est la commande que vous prononcez.
    • ttsName qui est le retour annoncé par SARAH.
  • je vous recommande de ne pas toucher aux autres paramètres.

    • 4. QUELLES SONT LES COMMANDES POSSIBLES :

    • pour importer la configuration : "Mets à jour la domotique"
    • pour les actionneurs : "Allumer", "Ouvre", "Active", "Eteint", "Ferme", "Désactive"
    • pour les sonde (temperature, hydro, luminosité) : "Comment est..."
    • pour les scenario : "lance le scénario..."
    • ... le tout précédé bien sûr de "SARAH..."
